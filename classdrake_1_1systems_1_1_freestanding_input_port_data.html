Detailed Description

template<typename T>
class drake::systems::FreestandingInputPortData< T >

The FreestandingInputPortData encapsulates a vector of data for use as an input to a System.

Template Parameters

VectorInterface<T>* GetMutableVectorData ( )
inline

Returns a pointer to the data inside this InputPortData, and updates the version so that Contexts depending on this InputPortData know to invalidate their caches.

To ensure invalidation notifications are delivered, callers should call this method every time they wish to update the stored value. In particular, callers MUST NOT write on the returned pointer if there is any possibility this FreestandingInputPortData has been accessed since the last time this method was called.
