Parses a model and adds it to the rigid body tree.

Parameters
rigid_body_treeA pointer to the rigid body tree to which to add the model.
nodeThe XML node containing the model information.
package_mapA map containing information about the ROS workspace in which to search for meshes.
root_dirThe root directory from which to search for mesh files.
floating_base_typeThe type of floating joint to use to weld the newly added model to the rigid body tree.
weld_to_frameSpecifies the initial pose of the newly added robot relative to the link to which the robot is being welded.
