Detailed Description

ValueListenerInterface is an interface that dependent computations must implement so that they can receive notifications when a prerequisite value is modified.

The prerequisite may be an independent value, like time or a state variable, or it may itself be a computation, which computation might be local or delivered through an input port.

This interface is a Drake-internal detail. Users should not implement it.
