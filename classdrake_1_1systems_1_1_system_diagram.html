Detailed Description

template<typename T>
class drake::systems::SystemDiagram< T >

A SystemDiagram is a still-abstract System that contains other System objects as subsystems, likely with internal interconnections.

A SystemDiagram's state is a composite of its subsystems' states, and its Context includes sub-Context objects as needed by its subsystems.

Template Parameters
TThe type of numerical values in this SystemDiagram. Must be a valid Eigen scalar.
