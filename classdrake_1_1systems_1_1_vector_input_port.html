Detailed Description

template<typename T>
class drake::systems::VectorInputPort< T >

A VectorInputPort extends InputPort with the restriction that only VectorOutputPort connections are acceptable, and the value will be of type VectorInterface.

You may optionally require that only a specified length of vector is acceptable; otherwise any length is allowed.
